PREREQUISITE:
  - Microsoft SQL Server Management Studio 2008 or above
  - Microsoft Visual Studio 2010 or above, and C# language package installed
  - Excel file used for importing must satisfy:
    + in .xlsx format
    + have only one sheet, name Sheet1
    + no cells are merged or splitted
    + the first row of the table, which contains column names in database, must have the same spelling with relation's attribute name


Step 1: Install AccessDatabaseEngine, if can not install, find another version on the Internet and install again

Step 2: Open SQL Server Management Studio, connect to server, and run ConfigurationsForImportingExcelFiles scripts

Step 3: Open PrimaryHorizontalFragmentation app
